Use Image in HTML Page

The <img> tag includes an image into the html page and change the width and height.

The default way to set the horizontal alignment of image is to use 'align' property. The default alignment is to the left.

In order to mix images and text into the layout together, you can also set the 'float' style to the <img> tag.

Images can also be used as a hyperlink. Just wrap the <a> tag outside.

In case you want an image to be your webpage background, set the 'background-image' style of the <body> tag using :url('imagename.jpg'). And you can also set the 'background-repeat' and 'background-position'. Howerver, be really careful with the image contrast to the content text.
